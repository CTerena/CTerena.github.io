使用MATLAB求解微积分问题

2021-10-24 2 min read

Part 1.极限问题的解析解

极限问题在MATLAB符号运算工具箱中可以使用limit()函数直接求出，其调用格式为

L = limit(f, x, x0) #求极限
L = limit(f, x ,x0,'left' 或 'right') #求单边极限

在求解之前我们需要先用语句syms x声明自变量x，再用符号表达式的形式定义原函数f，若x0为无穷，则可以用inf或-inf来表示。如果需要求解左右极限问题，还需要给出'left'或'right'选项。
如果函数中只有一个符号变量，则可以在调用语句中忽略该变量，即省略第二项。

示例：
当x从右侧趋于0时，求解y的极限
y=−ex3−1cos⁡(x−sin⁡(x))−1y=-\frac{{\mathrm{e}}^{x^3}-1}{\cos\left(\sqrt{x-\sin\left(x\right)}\right)-1}y=−cos(x−sin(x)​)−1ex3−1​

syms x;
f = (exp(x^3)-1)/(1-cos(sqrt(x-sin(x))));
limit(f, x, 0, 'left')

Part 2.函数导数的解析解

如果函数和自变量都已知，且均为符号变量，则可以用diff()函数接触给定函数的各阶导数。diff()函数的调用形式是f1 = diff(f,x,n),其中，f为给定函数，x为自变量，这两个变量均为符号型变量，n为导数的阶次，若省略n则默认为求一阶导数，若f表达式中只有一个变量，则可以省略变量x。

示例：

求解步骤：

  1. 声明x为符号变量
  2. 用MATLAB语句描述原函数
  3. 调用diff()函数求导

代码：

syms x;
f(x) = sin(x) / x^2 + 4*x + 3;
f1 = diff(f)
